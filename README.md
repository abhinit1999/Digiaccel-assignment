
# Quiz App MERN




## Run Locally

Clone the project

```bash
  git clone https://github.com/abhinit1999/Digiaccel-assignment
```

Go to the project directory

```bash
  cd Quiz-App-MERN
```

Install dependencies for Client And Server

```bash
  cd server
  npm install

  cd client
  npm install
```

Start the both server on seprate terminals

```bash
  cd server
  npm app.js
```

```bash
  cd client
  npm start
```
